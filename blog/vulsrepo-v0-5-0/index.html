VulsRepo を v0.5.0 に更新しました。WordPress 本体、テーマ、プラグインの脆弱性情報を表示できるようになりました。

更新内容

追加

  • #201 wpscan で WordPress 本体、テーマ、プラグインの脆弱性情報を表示できるようになりました。
  • #201 WordPress 向けに VulnType, Status と Update をピボットテーブルに追加しました。
項目 説明
VulnType (WordPress のみ) WPScan が提供している脆弱性のタイプ
Status (WordPress テーマ、プラグインのみ) WordPress テーマ、プラグインが有効かどうか
Update (WordPress テーマ、プラグインのみ) WordPress テーマ、プラグインのアップデートが利用できるかどうか
