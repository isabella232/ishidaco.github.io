VulsRepo v0.2.7 makes it easier to narrow down the data to be displayed

December 21, 2020

2 minutes read

 Twitter

 Facebook

 Reddit

 LinkedIn

 Pinterest

VulsRepo online demo v.0.2.7 is here.

v0.2.7

Features

  • #178 Added the ability to select data by specifying a date range and server (container).

For example, to display a specific server (container) for the last month, you had to expand the date tree one by one and select the target server repeatedly for the number of days.

With this improvement, it is now possible to do so by simply selecting Last Month and choosing the target server from the server list. You can also select individual servers from the tree as before.

The date range can be specified by selecting from the drop-down menu. Of course, you can also specify any date range.

You can select multiple target servers (containers) from the server list. You can also filter by server name and then select or deselect them.

Changes

  • #178 Changed so that the data of the server (container) with the most recent date is initially selected.

Fixes

  • #177 Fixed resultsPath so that it works even if the value ends with /.
