VulsRepo online demo v.0.2.4 is here.

v0.2.4

Features

  • #121 RedHatAPI information is now viewable.
  • #70 Mitigation information is now viewable.
  • #101 Pivot table settings can now be shared by URL.
  • #113 ConfidenceScore has been added to the pivot table.

Changes

  • #111 Changed evaluation values for CVSSv3 (*) and CVSS (*) to affect CVSS chart drawing.
  • #110 Change CVSSv3 (*) and CVSS (*) metrics sort order by decending value.
  • #131 Control display of USCERT / JPCERT Alert and Exploits sections in the Details Panel. Sections are now only shown when there is relevant information.
  • #104 Changed notation; Select all has been changed to Select All, and Deselect all has been changed to Select None.
  • #99 Changed filter notation from setting to filter.

Fixes

  • #118 Preset Filter: 05. Fixed Pivot: CveID/PackageInfo => NotFixedYet column from ScanTime to NotFixedYet and added Repository, FixedIn, and FixState to row.
  • #102 Fixed an issue where the timing of redrawing a pivot table would cause unsaved filters that were being applied to be removed.
  • #108 Prevented changing or deleting preset filters.
  • #44 Fixed an issue where the 11th and subsequent NotFixedYet would not be colored in the package list in the details panel.
  • #122 Fixed an issue where the package list in the details panel could not display the changelog panel after the 11th one.
  • #125 Fixed an issue where heatmaps in pivot tables would sometimes be deep in color.

docker pull ishidaco/vulsrepo:version-0.2.4
